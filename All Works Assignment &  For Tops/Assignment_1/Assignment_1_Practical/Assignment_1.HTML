1.HTML

HTML is the standard markup language for creating web pages. It structures content using elements and tags, enabling developers to design interactive and visually appealing websites. Essential for web development, HTML works seamlessly with CSS and JavaScript, forming the foundation of modern web design and user experiences.

2.CSS

CSS (Cascading Style Sheets) is a style sheet language used to control the appearance and layout of web pages. It allows developers to apply styles like colors, fonts, spacing, and positioning to HTML elements, ensuring consistency and responsiveness across devices, enhancing the visual appeal and usability of websites.
